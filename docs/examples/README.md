# Example PPTs

This folder contains **reference capability decks** that the agent uses to understand your company's tone, style, and structure.

## Purpose

When you ask the agent to build a new PPT, it reads decks stored here to:
- Match the slide structure you prefer
- Use language and framing that fits your consultancy style
- Understand which sections are most important
- Copy visual design patterns from your master template

## What to Add

Upload 2–3 existing capability decks you're proud of:
- Data & Analytics deck
- Cloud Migration deck
- AI/ML Services deck
- etc.

**Format**: `.pptx` files only

**Naming**: Clear names like:
```
Data-Analytics-Capability.pptx
Cloud-Migration-Capability.pptx
AI-ML-Services-Capability.pptx
```

## How the Agent Uses These

The agent will:
1. Read the structure (how many slides, what sections, order)
2. Extract the tone and language patterns
3. Understand which graphic elements are recurring
4. Apply the same patterns to new decks

This is **optional** — the agent works fine without examples. But with examples, it produces better results faster.
