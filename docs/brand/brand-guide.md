# Brand Guide

## Color Palette

### Dark Mode Colors

| Name   | Hex       | RGB             | Usage                          |
|--------|-----------|-----------------|--------------------------------|
| SEA    | `#00F6FF` | 0, 246, 255     | Highlight accents, dividers    |
| MINT   | `#00FFF0` | 0, 255, 240     | Secondary accents              |
| LILAC  | `#B896FF` | 184, 150, 255   | Subtle accent / callout boxes  |
| SKY    | `#7BA8FF` | 123, 168, 255   | Supporting accent              |
| NIGHT  | `#060606` | 6, 6, 6         | Text on light backgrounds      |

### Primary Light Mode Colors

| Name   | Hex       | RGB             | Usage                                      |
|--------|-----------|-----------------|--------------------------------------------|
| COBALT | `#0047FF` | 0, 71, 255      | **Primary brand color** — title bars, CTAs |
| AQUA   | `#0078C2` | 0, 120, 194     | **Secondary color** — headers, icons       |
| IRIS   | `#8453D2` | 132, 83, 210    | **Accent / highlight** — callouts, tags    |
| SLATE  | `#4A71BD` | 74, 113, 189    | Supporting icons, sub-headers              |
| SNOW   | `#FBFAFA` | 251, 250, 250   | **Background** — slide backgrounds        |

## Typography

| Element            | Font         | Size  | Weight  |
|--------------------|--------------|-------|---------|
| Slide title        | Calibri      | 28pt  | Bold    |
| Section heading    | Calibri      | 22pt  | Bold    |
| Body / bullets     | Calibri      | 14pt  | Regular |
| Caption / footnote | Calibri      | 10pt  | Regular |

> **Note**: Update font names here if the company uses a custom typeface.

## Logo

- File: `docs/brand/EPAM_LOGO_Black.png`
- Placement: top-right corner on Cover and Back Cover slides
- Minimum clear space: 16px on all sides
- Never stretch, recolor, or place on a busy background

---

## Slide Color Rules

- **Title bar background**: COBALT `#0047FF`
- **Title bar text**: SNOW `#FBFAFA`
- **Slide background**: SNOW `#FBFAFA`
- **Body text**: NIGHT `#060606`
- **Accent shapes / dividers**: AQUA `#0078C2`
- **Callout boxes / highlights**: IRIS `#8453D2` at 15% opacity
- **Charts — primary series**: COBALT `#0047FF`
- **Charts — secondary series**: AQUA `#0078C2`
- **Charts — tertiary series**: IRIS `#8453D2`
